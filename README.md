# Multiplying-two-matrix

## AIM:
Mutiplying tow matrix
## ALGORITHM:

### Step 1:
use import numpy
### Step 2:
Enter the input
### Step 3:
use append()
### Step 4:
use the multiply two matrix
### Step 5:
print the output
## PROGRAM: 
import numpy as np
l1,l2=[],[]
n=int(input())
for i in range(n):
    l1.append(int(input()))
for j in range(n):
    l2.append(int(input()))
array_1=np.array(l1)
array_2=np.array(l2)
product=array_1*array_2
print('Product of two arrays is:',product)
